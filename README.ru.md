# Users

Модуль `Users` управляет учётными записями в `osysHome`.

Через него администратор может:

- создавать и удалять пользователей;
- назначать роли `Guest`, `User`, `Editor`, `Administrator`;
- задавать домашнюю страницу и часовой пояс;
- генерировать и менять `API key`;
- загружать аватар;
- устанавливать или сбрасывать пароль.

В интерфейсе модуля доступен список пользователей с ролью, домашней страницей и временем последнего входа.
