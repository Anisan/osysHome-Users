# Users

The `Users` module manages accounts in `osysHome`.

Administrators can use it to:

- create and delete users;
- assign `Guest`, `User`, `Editor`, and `Administrator` roles;
- define home page and timezone;
- generate and rotate `API key` values;
- upload avatars;
- set or reset passwords.

The module interface includes a users list with role, home page, and last login time.
